# PPUR

## Pick
* You pick the repo, database, or marker source you want to inspect.

## Pull
* You pull a tool, script, or memory trace from the source into local
  or active memory.

## Use
* You use the tool, marker, or beat according to its intended design
  — it's already shaped for purpose.

## Release
* You release the tool from memory or runtime — clean up, discard, or
  destroy it to reset your environment.

---

"Pick what you need. Pull it into view. Use it with purpose. Release when through."
